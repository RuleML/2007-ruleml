International RuleML Symposium

on Rule Interchange and Applications

The International RuleML Symposium on Rule Interchange and Applications (RuleML-2007) is devoted to practical distributed rule technologies and rule-based applications which need language standards for rules operating in the context of, e.g., the Semantic Web, Intelligent Multi-Agent Systems, Event-Driven Architectures and Service-Oriented Computing Applications.
